# SimSolarSystem

Runge-Kutta numerical simulation of our Solar System.
Currently includes planets, moons, and 20 objects in the asteroid belt.

Should be run in this order:
 * populateKepler.py
 * convertToCartesian.py
 * system.py
 * draw.py
